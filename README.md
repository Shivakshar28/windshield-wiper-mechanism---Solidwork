# Windshield Wiper Mechanism | SolidWorks

## Overview

This repository contains the complete CAD design and assembly of an automotive windshield wiper mechanism developed using **SolidWorks**. The project demonstrates the design, modeling, and simulation of a mechanical linkage system that converts continuous rotary motion from an electric motor into the oscillatory motion required to sweep a vehicle's windshield. It serves as a practical application of mechanical design principles, linkage kinematics, and parametric CAD modeling.

## Project Objectives

The primary objective of this project is to understand the working principle of a windshield wiper mechanism and accurately model its components in a 3D CAD environment. The project focuses on creating a fully functional assembly while ensuring proper motion, alignment, and manufacturability. It also aims to strengthen skills in mechanical design, assembly creation, engineering documentation, and motion analysis.

## Mechanism Description

The windshield wiper mechanism is based on a linkage system that transforms the rotary motion generated by an electric motor into a controlled reciprocating motion of the wiper arm. This conversion enables efficient wiping across the windshield while maintaining a consistent sweep angle. A motion study was performed to verify the movement of the linkage, identify any interference between components, and ensure smooth operation throughout the mechanism's range of motion.
